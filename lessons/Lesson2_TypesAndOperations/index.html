Унарные арифметические операции

  • префиксный инкремент ( ++value )
  • постфиксный инкремент ( value++ )
  • префиксный декремент ( --value )
  • постфиксный декремент ( value-- )

Унарные арифметические операции

            int a = 8;
            int b = ++a; // a = 9, b = 9
            a = 8; b = a++; // a = 9, b = 8
            a = 8; b = --a; // a = 7, b = 7
            a = 8; b = a--; // a = 7, b = 8

Бинарные арифметические операции

  • вычитание («-»)
  • сложение («+»)
  • умножение («*»)
  • деление («/»)
  • остаток от деления («%»)

Бинарные арифметические операции

            int a = 10; int b = 7;
            int i = a / b;// 1
            double k = 10 / 4;// 2
            double k = 10.0 / 4;// 2.5
            int j = a % b;// 3

Приоритет арифметических операций

  • ++ (инкремент), -- (декремент)
  • * (умножение), / (деление), % (остаток от деления)
  • + (сложение), - (вычитание)

Приоритет арифметических операций

Пример: 
            int a = 8; int b = 7;
            int i = a / b;// 1
            int c = a + 5 * ++b; // 48
            int d = (a + 5) * ++b; // 104

Операции с присваиванием

Пример: 
            int c = 3;
            c += 5; // 8
            c -= 2; // 6
            c /= 3; // 2;
