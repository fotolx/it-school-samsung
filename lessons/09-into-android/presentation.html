Устройства на базе Android не запускают файлы .class и .jar

Вы не сможете воспользоваться обычной средой разработки на языке Java — вам понадобятся специальные инструменты для преобразования откомпилированного кода в формат Android

Все необходимое содержится в Android SDK

Android Studio — версия IDEA, которая включает версию Android SDK и дополнительные инструменты графических интерфейсов, упрощающие разработку приложений.

Android Studio предоставляет шаблоны, упрощающие создание новых приложений и классов, а также средства для выполнения таких операций, как упаковка приложений и их запуск.
