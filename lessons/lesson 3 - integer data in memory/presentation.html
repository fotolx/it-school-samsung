Хм... А если использовать экспоненциальную запись чисел?

Экспоненциальная запись действительных чисел

Любое вещественное число A может быть записано в экспоненциальной форме: A = ±m⋅qp , где:
m — мантисса числа;
q — основание системы счисления;
р — порядок числа.

Экспоненциальная запись действительных чисел

Например, число 472000000 может быть представлено так:
472000000 = 4,72⋅108
472000000 = 47,2⋅107
472000000 = 472,0⋅106
