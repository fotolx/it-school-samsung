Представление целочисленных типов данных

Данные примитивных типов представляются фиксированным количеством двоичных разрядов
Это очень важно, потому что задает ограничение на размер чисел
Например, если использовать всего три двоичных разряда, можно использовать только 8 различных значений: 000, 001, 010, 011, 100, 101, 110 и 111. То есть представлять 8 различных чисел.
